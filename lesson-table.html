TABLE LESSON

tags

  • <table></table>  = declare for creating table
  • <thead></thead>  = define content in header/li>
  • <tbody></tbody>  = contend in body
  • <tfoot></tfoot>  = content in bottom of table
  • <tr></tr>  = define row
  • <td></td>  = define colum
  • <th></th>  = column in header

attribute

  • border="0"    ===> table border
  • width="%"    ===> width
  • bgcolor="white"    ===> background color

structure
